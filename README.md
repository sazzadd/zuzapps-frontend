# Zuzapps Website

Preview Live URL: https://sazzadd.github.io/zuzapps-frontend/

Zuzapps is a modern CRM landing page website designed and developed for a Fiverr client project. The website presents Zuzapps as a smart CRM platform that helps teams manage relationships, automate workflows, track pipelines, and improve business operations from one clean workspace.

## Project Overview

This project is a responsive landing page built with HTML, Tailwind CSS, Swiper JS, and custom JavaScript. The design focuses on a clean SaaS style layout, strong hero section, CRM dashboard visuals, feature sections, customer testimonials, industry result cards, CTA section, and footer navigation.

## Features

=> Responsive landing page layout  
=> Modern SaaS style UI  
=> Sticky header with navigation  
=> Mobile friendly menu  
=> Hero section with CTA buttons  
=> Infinite logo slider  
=> Feature sections for CRM capabilities  
=> Testimonial slider using Swiper JS  
=> Business result cards  
=> Platform overview section  
=> Statistics cards  
=> Final CTA section  
=> Footer with dynamic copyright year

## Technologies Used

=> HTML5  
=> Tailwind CSS
=> JavaScript  
=> Swiper JS
